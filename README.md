 🚗 AI Road Accident Severity Prediction

## 📌 Overview

This project is an **AI-based Road Accident Severity Prediction System** that uses Machine Learning to predict the severity of road accidents based on various real-world factors such as location, weather, road type, and number of vehicles.

The model analyzes input data and classifies accidents into different severity levels, helping in improving road safety and decision-making.

---

## 🎯 Features

* Predict accident severity (Low / Medium / High)
* Uses real-world parameters:

  * Latitude & Longitude
  * Number of Vehicles
  * Number of Casualties
  * Day of the Week
  * Weather Conditions
  * Light Conditions
  * Road Type
  * Speed Limit
* Data visualization using graphs
* Clean Python implementation (.py)
* Stable execution without notebook issues
